# Pharmaceutical Sales Data Analysis

This project analyzes pharmaceutical sales data using Python, Pandas, Matplotlib, and Seaborn.

## Objectives
- Analyze total sales by drug category
- Identify top-selling drug categories
- Compare sales across different periods
- Analyze average daily sales
- Explore seasonal trends in respiratory drugs (R03)

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Pharmaceutical sales dataset containing daily sales information for various ATC drug categories.

## Key Findings
- Identified top-selling drug categories.
- Compared sales performance across different periods.
- Determined the highest average daily sales category.
- Analyzed seasonal trends in respiratory drug sales.
